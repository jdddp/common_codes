检测结果对比工具
=============================================================

功能
----
用于比较浮点模型(FP32)与量化模型(INT8)的检测结果差异。

输入：
    两个保存检测结果(JSON)的文件夹

输出：
    - 检测目标数量
    - 匹配目标数量
    - 漏检(Missing)
    - 多检(Extra)
    - 平均 IoU
    - 平均 Score 误差
    - Bounding Box 偏移(dx/dy/dw/dh)

-------------------------------------------------------------
JSON 格式
-------------------------------------------------------------

{
    "image": "000001.jpg",
    "objects":
    [
        {
            "label":"person",
            "label_id":0,
            "score":0.95,
            "bbox":[100.5,120.2,50.3,80.4]
        }
    ]
}

bbox 格式：

[x, y, width, height]

坐标均为映射回原图后的坐标。

-------------------------------------------------------------
目录结构
-------------------------------------------------------------

results_fp32/
    000001.json
    000002.json
    ...

results_int8/
    000001.json
    000002.json
    ...

要求：

1、两个目录中文件名必须一致

例如：

results_fp32/
    test001.json

results_int8/
    test001.json

否则无法进行比较。

-------------------------------------------------------------
运行方式
-------------------------------------------------------------

python compare_quant.py \
    --dir1 ./results_fp32 \
    --dir2 ./results_int8


-------------------------------------------------------------
评价指标说明
-------------------------------------------------------------

Matched
    成功匹配的目标数量。

Missing
    FP32 存在但 INT8 未匹配到的目标数量。

Extra
    INT8 多出来的目标数量。

Mean IoU
    匹配目标框平均 IoU。

Mean Score Error
    confidence 平均误差：

        abs(score_fp32-score_int8)

Mean dx/dy/dw/dh

    Bounding Box 平均偏移：

        dx = |x_fp32-x_int8|
        dy = |y_fp32-y_int8|
        dw = |w_fp32-w_int8|
        dh = |h_fp32-h_int8|

数值越小表示量化误差越小。
